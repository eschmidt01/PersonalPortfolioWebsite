Personal Portfolio Website

Eric Schmidt - Student #: 720309

This web app offers a professional and engaging showcase of skills, projects, and accomplishments, with interactive features like a project gallery, testimonials, a downloadable resume, and contact options for networking.

Desired Outcomes

  1. Professional Presentation
  2. Showcase of Skills and Projects
  3. Engaging User Experience
  4. Easy Access to Resume
  5. Accessible Design
  6. Personal Branding

This section provides a brief introduction to the portfolio owner, highlighting key skills, experiences, and interests. It sets the tone for the rest of the website and establishes a personal connection.

Relevant Outcomes: Professional Presentation, Personal Branding

  • Biography

  • Key Skills
  • Social Media Links

Example and Short Description:
GitHub Repository: Responsive Portfolio About Section
